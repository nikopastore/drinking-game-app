# SipWiki App Store Optimization (ASO) Guide

## Overview

This guide contains optimized metadata for Apple App Store and Google Play Store submissions. The strategy prioritizes "party/social" keywords over "drinking" keywords to maximize reach while minimizing policy risk.

---

### App Name (30 characters max)
```
SipWiki - Party Games
```

### Subtitle (30 characters max)
```
Icebreakers & Group Fun
```

```
party games,icebreaker,group games,adult games,pregame,drinking games,beer pong,kings cup,truth dare
```

## Keyword Strategy

### Tier 1: Primary Focus (High Volume, Lower Competition)
| Keyword | Search Volume | Competition |
|---------|--------------|-------------|
| party games for adults | High | Medium |
| icebreaker games | High | Low |
| group games for adults | Medium | Low |
| pre-game app | Medium | Low |
| house party games | Medium | Low |

### Tier 2: Activity-Based (Capture Specific Intent)
| Keyword | Search Volume | Competition |
|---------|--------------|-------------|
| kings cup rules | High | Medium |
| beer pong rules | High | Medium |
| never have i ever game | High | Medium |
| truth or dare app | High | High |
| flip cup rules | Medium | Low |

### Tier 3: Contextual (High Conversion)
| Keyword | Search Volume | Competition |
|---------|--------------|-------------|
| drinking game rules | High | High |
| party game app | Medium | Medium |
| game night ideas | Medium | Low |
| fun group activities | Medium | Low |

### Keywords to AVOID (Policy Risk)
- "get drunk"
- "wasted"
- "binge drinking"
- "shots game"
- "alcohol game"
- "buzzed"

## Visual Assets Checklist

### App Icon
- [ ] Bright, recognizable at small sizes
- [ ] No alcohol imagery (bottles, glasses)
- [ ] Party/fun vibe (could use: cards, dice, people, celebration)

### Screenshots (6-10 required)
1. [ ] Hero shot: Game library with colorful cards
2. [ ] Feature: Spin the wheel in action
3. [ ] Feature: Game rules page (Kings Cup)
4. [ ] Feature: Filter options
5. [ ] Social proof: "30+ Games" callout
6. [ ] Lifestyle: People laughing (illustrated or photo)

### Screenshot Text Overlays
- "30+ Party Games"
- "Complete Rules for Every Game"
- "Can't Decide? Spin the Wheel!"
- "Find the Perfect Game"
- "Works Offline"
- "No Ads, Just Fun"

### Preview Video (15-30 seconds)
- [ ] Show real people reacting/laughing (UGC style)
- [ ] Quick cuts of app interface
- [ ] Focus on FUN, not drinking
- [ ] End with app icon and name

1. **English (US)** - Primary
2. **English (UK)** - Minor tweaks (uni vs college)
3. **Spanish** - Large market
4. **German** - Strong drinking game culture
5. **French** - European reach

- **Rating**: 17+ (for alcohol references)

- **Rating**: Mature 17+


### Feature Request
```

```

### Bug Report
```
Sorry about that! Can you email us at [support email] with details? We'll fix it ASAP.
```

### Negative (No Specific Issue)
```
We'd love to make SipWiki better for you! What games or features would you like to see?
```
